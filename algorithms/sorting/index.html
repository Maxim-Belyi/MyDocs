Сортировки (Sorting)

В реальной разработке на Go вам практически никогда не придется писать алгоритмы сортировки вручную — стандартные пакеты sort и slices справляются с этим великолепно. Тем не менее, на алгоритмических секциях могут попросить реализовать базовые алгоритмы, чтобы проверить ваше понимание Computer Science.

1. Сортировка в стандартной библиотеке Go​

До Go 1.21 использовался пакет sort. Под капотом sort.Slice использует алгоритм pdqsort (Pattern-Defeating Quicksort), который имеет сложность $O(N \log N)$ в среднем и $O(N)$ в лучшем случае (если массив уже отсортирован).

import "sort"

func main() {
    nums := []int{5, 2, 9, 1, 5}
    sort.Ints(nums) // Для базовых типов есть готовые хелперы
    
    // Кастомная сортировка (по убыванию)
    sort.Slice(nums, func(i, j int) bool {
        return nums[i] > nums[j]
    })
}

С появлением дженериков (Go 1.21+) появился более удобный и быстрый пакет slices:

import "slices"

func main() {
    nums := []int{5, 2, 9, 1, 5}
    slices.Sort(nums)
    
    // Пользовательская функция сортировки (cmp.Compare)
    slices.SortFunc(nums, func(a, b int) int {
        return b - a // По убыванию
    })
}

2. Классические алгоритмы (для собеседований)​

Быстрая сортировка (Quick Sort)​

Самый популярный алгоритм на собеседованиях. Идея: выбрать опорный элемент (pivot), разбить массив на элементы меньше опорного и больше опорного, а затем рекурсивно отсортировать эти две части.

Сложность: $O(N \log N)$ в среднем, $O(N^2)$ в худшем случае (если pivot всегда выбирается неудачно).

func quickSort(arr []int) []int {
    if len(arr) < 2 {
        return arr
    }
    
    // Выбираем опорный элемент (для простоты — средний)
    pivot := arr[len(arr)/2]
    
    var left, middle, right []int
    
    for _, item := range arr {
        if item < pivot {
            left = append(left, item)
        } else if item > pivot {
            right = append(right, item)
        } else {
            middle = append(middle, item)
        }
    }
    
    // Рекурсивно сортируем левую и правую части, затем объединяем
    left = quickSort(left)
    right = quickSort(right)
    
    result := append(left, middle...)
    result = append(result, right...)
    return result
}

(Примечание: это наглядная, но не самая оптимальная in-place реализация. In-place QuickSort требует сложной логики перестановок индексов left и right).

Сортировка слиянием (Merge Sort)​

Идея: рекурсивно разделить массив пополам до единичных элементов, а затем "слить" отсортированные части вместе. Это стабильная сортировка.

Сложность: строго $O(N \log N)$, но требует $O(N)$ дополнительной памяти для слияния.

func mergeSort(items []int) []int {
    if len(items) < 2 {
        return items
    }
    mid := len(items) / 2
    left := mergeSort(items[:mid])
    right := mergeSort(items[mid:])
    return merge(left, right)
}

func merge(left, right []int) []int {
    result := make([]int, 0, len(left)+len(right))
    i, j := 0, 0
    
    // Сливаем два отсортированных массива
    for i < len(left) && j < len(right) {
        if left[i] < right[j] {
            result = append(result, left[i])
            i++
        } else {
            result = append(result, right[j])
            j++
        }
    }
    // Добавляем остатки
    result = append(result, left[i:]...)
    result = append(result, right[j:]...)
    return result
}

3. Сортировка подсчетом (Counting Sort)​

Невероятно быстрый алгоритм $O(N)$, который не использует сравнения, но применим только когда мы знаем, что числа находятся в ограниченном диапазоне (например, от 0 до 100).

Идея: создать массив-счетчик размером с максимальное число, пройтись по данным и подсчитать количество каждого элемента, а затем восстановить массив.

func countingSort(arr []int, maxValue int) {
    counts := make([]int, maxValue+1)
    
    // Подсчитываем частоту
    for _, num := range arr {
        counts[num]++
    }
    
    // Восстанавливаем массив
    index := 0
    for num, count := range counts {
        for i := 0; i < count; i++ {
            arr[index] = num
            index++
        }
    }
}
