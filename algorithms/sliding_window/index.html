Плавающее окно (Sliding Window)

Плавающее окно (Sliding Window) — это алгоритмический паттерн, являющийся расширением паттерна "Два указателя". Он используется для задач, где нужно найти какую-то подпоследовательность (подмассив или подстроку) в линейной структуре данных, удовлетворяющую определенным условиям.

Смысл алгоритма в том, чтобы представить подмассив как "окно" с левой и правой границей. Вместо того, чтобы каждый раз пересчитывать сумму/элементы внутри окна заново ($O(N^2)$), мы сдвигаем окно, добавляя элемент справа и удаляя элемент слева, достигая сложности O(N).

Вариант 1: Окно фиксированного размера​

Размер окна k задан заранее.

Задача: Максимальная сумма подмассива размера K​

Дан массив чисел и число K. Найти максимальную сумму смежных K элементов.

func maxSum(nums []int, k int) int {
    if len(nums) < k || k <= 0 {
        return 0
    }
    
    // 1. Считаем сумму для первого окна
    currentSum := 0
    for i := 0; i < k; i++ {
        currentSum += nums[i]
    }
    
    maxSum := currentSum
    
    // 2. Сдвигаем окно вправо на 1 элемент
    for i := k; i < len(nums); i++ {
        // Добавляем новый правый элемент и вычитаем старый левый
        currentSum = currentSum + nums[i] - nums[i-k]
        
        if currentSum > maxSum {
            maxSum = currentSum
        }
    }
    
    return maxSum
}

Вариант 2: Окно динамического размера​

В этом случае размер окна не фиксирован. Окно расширяется (сдвиг правой границы), пока выполняется определенное условие. Если условие нарушается, окно начинает сужаться (сдвиг левой границы), пока условие снова не станет верным.

Задача: Самая длинная подстрока без повторяющихся символов​

Классическая задача с LeetCode (#3). Найти длину максимальной подстроки, где все символы уникальны.

func lengthOfLongestSubstring(s string) int {
    // Храним символ и количество раз, которое он встретился в текущем окне
    charCounts := make(map[byte]int) 
    
    left := 0
    maxLength := 0
    
    for right := 0; right < len(s); right++ {
        char := s[right]
        charCounts[char]++ // Добавляем правый символ в окно
        
        // Если символ повторился, значит окно стало невалидным.
        // Начинаем сужать окно слева, пока повтор не исчезнет.
        for charCounts[char] > 1 {
            leftChar := s[left]
            charCounts[leftChar]--
            left++ // Сдвигаем левую границу
        }
        
        // Окно снова валидно, проверяем длину
        currentLength := right - left + 1
        if currentLength > maxLength {
            maxLength = currentLength
        }
    }
    
    return maxLength
}

Как узнать, что нужен Sliding Window на собеседовании?​

Ищите в условии задачи следующие маркеры:

  1. Дан массив (или строка).
  2. Просят найти подмассив, подстроку, или непрерывную последовательность (contiguous subarray).
  3. Просят найти самую длинную, самую короткую, максимальную сумму или минимальное значение.

Sliding Window — это идеальный инструмент для превращения $O(N^2)$ брутфорса в элегантный проход за $O(N)$.
