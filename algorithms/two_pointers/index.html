Два указателя (Two Pointers)

Два указателя (Two Pointers) — это алгоритмический паттерн, при котором используются два индекса (указателя) для прохода по структуре данных (обычно массиву, срезу или строке).

Обычно один указатель начинает с начала, а другой с конца, и они двигаются навстречу друг другу, либо оба двигаются в одном направлении, но с разной скоростью. Паттерн значительно снижает временную сложность с O(N²) (вложенные циклы) до O(N).

Вариант 1: Навстречу друг другу​

Чаще всего применяется на отсортированных массивах или когда нужно сравнить элементы с двух концов.

Задача: Палиндром​

Проверить, является ли строка палиндромом (читается одинаково слева направо и справа налево).

func isPalindrome(s string) bool {
    // Конвертируем в руны для корректной поддержки кириллицы и юникода
    runes := []rune(s)
    
    left := 0
    right := len(runes) - 1
    
    for left < right {
        if runes[left] != runes[right] {
            return false // Не совпало, не палиндром
        }
        left++
        right--
    }
    return true
}

Задача: Разворот среза (Reverse Array)​

Как эффективно развернуть массив [1, 2, 3, 4, 5] in-place (без выделения дополнительной памяти).

func reverse(nums []int) {
    left := 0
    right := len(nums) - 1
    
    for left < right {
        // Множественное присваивание в Go избавляет от нужды во временной переменной temp
        nums[left], nums[right] = nums[right], nums[left]
        left++
        right--
    }
}

Задача: Two Sum на отсортированном массиве​

Если массив уже отсортирован, задачу Two Sum можно решить без хэш-таблицы (без дополнительной памяти O(1)).

func twoSumSorted(nums []int, target int) []int {
    left, right := 0, len(nums)-1
    
    for left < right {
        sum := nums[left] + nums[right]
        
        if sum == target {
            return []int{left, right}
        } else if sum < target {
            left++  // Сумма слишком мала, берем бóльшее число (сдвигаем левый)
        } else {
            right-- // Сумма слишком велика, берем меньшее число (сдвигаем правый)
        }
    }
    return nil
}

Вариант 2: В одном направлении (Быстрый/медленный)​

Этот паттерн часто называют Fast and Slow pointers. Он полезен для фильтрации массивов in-place или обнаружения циклов в связанных списках (об этом в статье про Linked Lists).

Задача: Удаление дубликатов (Remove Duplicates)​

Дан отсортированный массив. Нужно удалить дубликаты in-place, чтобы каждый элемент встречался один раз, и вернуть новую длину массива.

func removeDuplicates(nums []int) int {
    if len(nums) == 0 {
        return 0
    }
    
    // slow указывает на последний уникальный элемент
    slow := 0 
    
    // fast ищет новые уникальные элементы
    for fast := 1; fast < len(nums); fast++ {
        if nums[fast] != nums[slow] {
            slow++
            nums[slow] = nums[fast] // Перезаписываем следующий элемент уникальным
        }
    }
    
    return slow + 1 // Количество уникальных элементов
}

Итог для собеседования​

Если вы слышите слова "отсортированный массив", "палиндром" или "in-place (О(1) по памяти)", первым делом подумайте, нельзя ли применить паттерн "Два указателя". В Go он реализуется через обычные переменные left и right, хранящие целочисленные индексы среза.
