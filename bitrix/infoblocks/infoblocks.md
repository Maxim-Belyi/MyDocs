# Инфоблоки (информационные блоки)

<span style="color: yellow">Инфоблоки</span> это модуль для структурированного хранения, управления и вывода однородных динамических данных, таких как товары, новости, фотогалереи или статьи. Они действуют как «база данных в базе данных», позволяя упорядочивать информацию по типам, разделам и элементам без необходимости менять физическую структуру базы данных SQL. 

# Основные характеристики инфоблоков:
* Структура: Состоят из типов (например, «Каталог» или «Контент»), самих инфоблоков (например, «Одежда», «Новости компании»), разделов (категорий) и элементов (конкретных товаров или статей).

* Свойства: К элементам можно добавлять неограниченное число дополнительных свойств (цена, артикул, фото, дата).

* Гибкость: Позволяют быстро менять структуру данных (добавлять/удалять поля) в административной панели.

* Вывод данных: Для отображения данных на сайте используются специальные компоненты, которые выводят информацию из инфоблоков.

* Инфоблоки 2.0: Улучшенная версия, оптимизирующая работу с множественными свойствами и ускоряющая выборку данных.

* Инфоблоки являются основой для большинства динамических разделов на сайтах под управлением «1С-Битрикс».
