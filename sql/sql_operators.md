# Операторы SQL

Операторы SQL — это ключевые слова и символы, позволяющие выполнять действия с данными и структурой базы данных, разделенные на группы: DDL (CREATE, ALTER, DROP для структуры), DML (SELECT, INSERT, UPDATE, DELETE для данных), DCL (GRANT, REVOKE для прав) и другие (арифметические `+, -, *, /; сравнения <, >, =;` логические AND, OR, NOT). Они являются основой для написания запросов, выбирая, изменяя, добавляя или удаляя данные. 


## Основные группы операторов
DDL (Data Definition Language) - Управление структурой БД:
* CREATE: Создание объектов (таблицы, представления).
* ALTER: Изменение существующих объектов.
* DROP: Удаление объектов.
____
 DML (Data Manipulation Language) - Работа с данными:
* SELECT: Извлечение данных.
* INSERT: Добавление новых записей.
* UPDATE: Изменение существующих записей.
* DELETE: Удаление записей.
____
 DCL (Data Control Language) - Управление доступом:
* GRANT: Предоставление прав.
* REVOKE: Отзыв прав. 

## Операторы сравнения (в WHERE)
* `= (равно), <> или != (не равно).`
* `> (больше), < (меньше).`
* `>= (больше или равно), <= (меньше или равно).` 

## Логические операторы (в WHERE)
* AND: Истина, если все условия истинны.
* OR: Истина, если хотя бы одно условие истинно.
* NOT: Отрицание (инвертирует результат).
* IS NULL: Проверка на NULL. 

## Арифметические операторы (в выражениях)
`+, -, *, / (сложение, вычитание, умножение, деление). `

## Другие полезные операторы
* ORDER BY: Сортировка результатов.
* GROUP BY: Группировка строк.
* LIKE: Поиск по шаблону в строках.
