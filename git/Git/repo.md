# Что такое репозиторий?
Репозиторий (часто сокращают до "репо") — это, по сути, папка с вашим проектом, за которой следит Git. Внутри этой папки есть скрытая подпапка `.git`, где Git хранит всю историю изменений, информацию о ветках, коммитах и другие служебные данные. Репозитории бывают двух видов:
*   **Локальный репозиторий:** Находится на вашем компьютере.
*   **Удалённый репозиторий:** Находится на сервере, например, на GitHub. Он используется для синхронизации работы между несколькими разработчиками и для резервного копирования.
