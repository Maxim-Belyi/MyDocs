# Для чего нужен `.gitignore`?


Файл `.gitignore` — это текстовый файл, в котором вы перечисляете файлы и папки, которые Git должен **игнорировать**. Такие файлы не будут отслеживаться, добавляться в коммиты и отправляться в удалённый репозиторий.

Это полезно для:
*   Файлов, которые генерируются автоматически (например, папка `node_modules`).
*   Секретных данных (пароли, ключи API).
*   Лог-файлов и временных файлов.
*   Файлов конфигурации операционной системы (`.DS_Store`, `Thumbs.db`).
