#### **3. Что такое коммит (commit)?**

Коммит — это "снимок" или точка сохранения состояния вашего проекта в определённый момент времени. Когда вы делаете коммит, вы фиксируете все подготовленные (staged) изменения и добавляете к ним краткое описание — сообщение коммита. Каждый коммит имеет уникальный идентификатор (хэш), и вся история проекта представляет собой цепочку последовательных коммитов.

Хорошая практика — делать небольшие, логически завершённые коммиты с понятными сообщениями. Например, "Добавил кнопку входа" вместо "Разные исправления".
