# fmt

Пакет `fmt` реализует форматированный ввод и вывод с функциями, аналогичными функциям `printf` и `scanf` языка C. Это один из наиболее часто используемых стандартных пакетов для работы с текстом в консоли или строками.

---

## 1. Вывод в консоль: Print, Println, Printf

Основная задача пакета — вывод данных на стандартный вывод (`os.Stdout`).

### Print и Println
Функции для базового вывода. `Println` автоматически добавляет пробелы между аргументами и символ новой строки в конце.

```go
package main

import "fmt"

func main() {
    name := "Go"
    version := 1.22
    
    fmt.Print("Язык: ", name, "\n")
    fmt.Println("Версия:", version)
}
```

### Printf
Используется для форматированного вывода с помощью "глаголов" (verbs).

```go
fmt.Printf("Язык: %s, Версия: %.2f\n", name, version)
```

> **%v** (value) — универсальный глагол, который выводит значение в формате по умолчанию. Используйте **%+v** для вывода названий полей структур и **%#v** для вывода синтаксического представления Go.

## 2. Форматирование строк: Sprint, Sprintf

Если вам нужно не вывести строку в консоль, а сохранить её в переменную, используются функции с префиксом `S`.

```go
message := fmt.Sprintf("Ошибка подключения к БД на порту %d", 5432)
// message теперь содержит отформатированную строку
```

Это стандартный способ конкатенации строк со значениями разных типов.

---

## 3. Вывод в потоки (io.Writer): Fprint, Fprintf

Эти функции аналогичны `Print` и `Printf`, но записывают отформатированные данные не в консоль, а в любой объект, реализующий интерфейс `io.Writer` (например, в открытый файл, сетевое соединение или HTTP-ответ).

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    file, _ := os.Create("report.txt")
    defer file.Close()
    
    // Запись отформатированной строки прямо в файл
    fmt.Fprintf(file, "Отчет за %d год\n", 2024)
}
```

В веб-разработке `fmt.Fprintf` часто используется для отправки отформатированного текста клиенту через `http.ResponseWriter`.

---

## 3. Работа с ошибками: Errorf

Функция `Errorf` позволяет создавать новые ошибки с форматированным текстом. Начиная с Go 1.13, она поддерживает глагол `%w` (wrap) для оборачивания других ошибок.

```go
import (
    "fmt"
    "os"
)

func readFile(filename string) error {
    _, err := os.Open(filename)
    if err != nil {
        return fmt.Errorf("ошибка при открытии файла %s: %w", filename, err)
    }
    return nil
}
```

---

## 4. Ввод данных: Scan, Scanf, Scanln

Функции для чтения данных со стандартного ввода (`os.Stdin`). На практике используются редко из-за ограничений в обработке пробелов и переносов строк.

```go
var name string
var age int

fmt.Println("Введите имя и возраст:")
fmt.Scan(&name, &age)
```

Для надежного консольного ввода чаще используют пакеты `bufio` и `os`.

---

## Итог: Чек-лист по пакету fmt

1.  **Простой вывод:** используйте `Println`.
2.  **Форматированный вывод:** используйте `Printf` и глагол `%v` для большинства задач.
3.  **Создание строк:** используйте `Sprintf` вместо множественной конкатенации `+`.
4.  **Создание ошибок:** используйте `Errorf` с параметром `%w` для сохранения контекста оригинальной ошибки.
