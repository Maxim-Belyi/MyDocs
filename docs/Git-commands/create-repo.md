### Вопросы и ответы: Основные команды Git

#### **1. Настройка и создание репозитория**

*   `git config --global user.name "Your Name"`
    `git config --global user.email "youremail@example.com"`
    **Что делает:** Устанавливает ваше имя и email, которые будут указываться в качестве автора для всех ваших коммитов. Флаг `--global` означает, что эта настройка будет применяться для всех ваших проектов на компьютере. Это первое, что нужно сделать после установки Git.

*   `git init`
    **Что делает:** Инициализирует (создаёт) новый Git-репозиторий в текущей папке. После выполнения этой команды в папке появится скрытая подпапка `.git`, где будет храниться вся история проекта.
    **Пример:** `git init my-project` — создаст папку `my-project` и инициализирует в ней репозиторий.

*   `git clone [url]`
    **Что делает:** Создаёт локальную копию удалённого репозитория, который находится по указанному `[url]`. Это самый частый способ начать работу над уже существующим проектом.
    **Пример:** `git clone https://github.com/docusaurus/docusaurus.git`
