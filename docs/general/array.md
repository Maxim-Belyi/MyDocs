# Что такое массив?

Массив — это упорядоченная структура данных, хранящая набор однотипных элементов (чисел, строк, объектов) под одним именем, к которым можно обращаться по уникальному номеру — индексу, который обычно начинается с нуля

##  Виды массивов

* Одномерный массив (вектор): Простой список, похожий на строку или столбец.

* Многомерные массивы (матрицы): Массивы массивов, используемые для представления таблиц (строки и столбцы).

## Основные операции

* Insert-вставляет элемент по заданному индексу

* Get-возвращает элемент по заданному индексу

* Delete-удаление элемента по заданному индексу

* Size-получить общее количество элементов в массиве
