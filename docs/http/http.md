#### **1. Что такое HTTP?**


HTTP (HyperText Transfer Protocol — протокол передачи гипертекста) — это фундаментальный протокол прикладного уровня, который используется для обмена данными в интернете. Он работает по модели "клиент-сервер". Проще говоря, когда вы вводите адрес сайта в браузере (клиент), браузер отправляет HTTP-запрос на сервер, где хранится сайт. Сервер обрабатывает этот запрос и отправляет обратно HTTP-ответ, который содержит запрошенные ресурсы (например, HTML-страницу, картинки, стили).
