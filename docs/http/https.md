# Что такое HTTPS и чем он отличается от HTTP?


HTTPS (HyperText Transfer Protocol Secure) — это расширение протокола HTTP, которое поддерживает шифрование для повышения безопасности. Ключевое отличие заключается в букве «S», которая означает «Secure» (безопасный).

Основные различия:
*   **Безопасность:** HTTP передает данные в открытом, незашифрованном виде. Это делает их уязвимыми для перехвата. HTTPS же шифрует данные с помощью криптографических протоколов SSL/TLS, обеспечивая их конфиденциальность и целостность.
*   **Порт:** HTTP по умолчанию использует порт 80, а HTTPS — порт 443.
*   **Наличие сертификата:** Для работы HTTPS сайту необходим SSL/TLS-сертификат, который подтверждает подлинность сайта и позволяет установить зашифрованное соединение.

Сегодня большинство сайтов используют HTTPS, так как ему доверяют и пользователи, и поисковые системы.
