#### ** Как работает HTTPS шифрование?**


HTTPS использует криптографический протокол SSL (Secure Sockets Layer) или его более современную версию TLS (Transport Layer Security) для защиты соединения.

Процесс работает так:
1.  **Проверка сертификата:** Когда вы подключаетесь к сайту по HTTPS, ваш браузер запрашивает у сервера его SSL/TLS-сертификат.
2.  **Аутентификация:** Браузер проверяет, что сертификат выдан доверенным центром сертификации и принадлежит именно этому сайту.
3.  **Обмен ключами:** Браузер и сервер используют информацию из сертификата (в частности, открытый ключ) для создания уникального сеансового ключа, которым будут шифроваться все данные во время этого визита.
4.  **Безопасная передача:** Вся информация между вашим браузером и сервером передается в зашифрованном виде, и расшифровать ее может только тот, у кого есть сеансовый ключ (то есть только ваш браузер и сервер).
