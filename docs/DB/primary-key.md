# Первичный ключ

Первичный ключ (Primary Key) — это столбец или набор столбцов в таблице базы данных, который однозначно идентифицирует каждую запись (строку), обеспечивая её уникальность. Он должен содержать уникальные значения и не может быть пустым (NULL). Первичный ключ позволяет быстро находить записи, связывать таблицы и поддерживать целостность данных, выступая "основой" для каждой строки.
